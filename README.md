🚀 Proyecto Fullstack: Onboarding Symfony & Angular

Este repositorio representa mi proceso de especialización en un entorno Fullstack profesional. A través de este proyecto base, he trabajado en la integración de un ecosistema moderno compuesto por un backend en Symfony y un frontend en Angular, gestionando todo el despliegue mediante contenedores Docker.


🛠️ Lo que he dominado en este proyecto:

    Entornos de Contenedores: Configuración y levantamiento de servicios (PHP, Nginx, PostgreSQL) usando Docker Compose.

    Arquitectura Backend: Exploración de la lógica de negocio y controladores en Symfony 7.

    Desarrollo Frontend: Gestión de componentes y servicios reactivos en Angular 17.

    Flujo de Trabajo Git Pro: Implementación de Conventional Commits y gestión de ramas mediante el flujo de Merge Requests.

    Estilos Modernos: Integración de Tailwind CSS para un diseño ágil y responsive.

📁 Estructura del proyecto

    /angular-frontend: Interfaz de usuario dinámica.

    /symfony-backend: API robusta y gestión de base de datos.

    /docs: Documentación personal y registro de participación.
